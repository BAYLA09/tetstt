# Design System

Premium Arabic beauty brand, not marketplace dropshipping.

Colors: cream, white, warm surface, dark oud text, muted taupe, gold, rose, success, danger.

Layout: max content width 1180-1240px, mobile padding 16px, desktop section padding 72-96px, alternate image/text sections on desktop.

Drawer/dialog: RTL layout, focusable close controls, scroll lock, sticky CTA at bottom.
